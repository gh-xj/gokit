## agentcli-go v0.2.3

Release date: 2026-02-24

### Highlights

- Added behavior-only regression framework for loop runs:
  - baseline snapshot write/read
  - drift comparison against committed baseline
  - CI gate wired via `task regression:behavior`
- Added profile-based quality policy files:
  - `configs/loop-profiles.json`
  - `configs/skill-quality.roles.json`
- Added skill quality judger executable:
  - `go run ./internal/tools/skillquality --context ... --repo-root ...`
- Added project-level skill packs:
  - `skills/agentcli-go`
  - `skills/loop-governance`
- Added Chinese README (`README.zh-CN.md`).

### Quickstart

```bash
go install github.com/gh-xj/agentcli-go/cmd/agentcli@v0.2.3
agentcli loop quality --repo-root .
agentcli loop regression --repo-root .
```

### Notes

- This release focuses on governance and regression stability for loop-driven workflows.
- Proxy propagation fallback for release/consumers: if proxy lag is observed, retry with `GOPROXY=direct` (for example, `GOPROXY=direct go install github.com/gh-xj/agentcli-go/cmd/agentcli@v0.2.3`).
