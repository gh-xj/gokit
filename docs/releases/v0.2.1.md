## agentcli-go v0.2.1

Release date: 2026-02-23

### Highlights

- Added `--description` to `agentcli add command`.
- Added preset-based scaffolding:
  - `--preset file-sync`
  - `--preset http-client`
  - `--preset deploy-helper`
- Added preset discovery:
  - `agentcli add command --list-presets`
- Generated command stubs now include preset-aware JSON/plain outputs.
- Onboarding docs updated with install verification and preset-first flow.

### Quickstart

```bash
go install github.com/gh-xj/agentcli-go/cmd/agentcli@v0.2.1
agentcli new --module example.com/mycli mycli
agentcli add command --dir ./mycli --preset file-sync sync-data
agentcli doctor --dir ./mycli --json
cd mycli && task verify
```

- This release improves time-to-first-usable-command while preserving deterministic scaffold and contract checks.
